### Jekyll本地环境安装
```bash
#  安装ruby
$ sudo apt-get install ruby
#  安装jekyll
$ sudo gem install jekyll
#  安装kramdown
$ sudo gem install kramdown
#  安装rouge
$ sudo gem install rouge
#  查看Jekyll版本
$ jekyll --version
```

### 本地运行
```bash
# 新建项目
$ jekyll new nick_blog
# 跑起来
$ jekyll server
```
