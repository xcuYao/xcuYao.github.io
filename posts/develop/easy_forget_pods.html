多版本cocoapods

cocoapods 是通过RubyGems来分发的，安装多版本的cocoapods可以通过安装不同版本的的ruby 来实现 rvm用来管理不同版本ruby

// 安装rvm
curl -L get.rvm.io | bash -s stable
// 查看可用的ruby版本信息
rvm list known
// 安装ruby2.5
rvm install 2.5
// 查看已经安装的ruby版本
rvm list
// 切换ruby版本
rvm use 2.5/default/2.4/2.6 ....
// 卸载已经安装的ruby版本
rvm remove 2.5
// 安装cocoapods指定版本
sudo gem install cocoapods -v 1.8.4
// cocoapods的所有版本
https://rubygems.org/gems/cocoapods/versions
// 卸载cocoapods
sudo gem uninstall cocoapods
